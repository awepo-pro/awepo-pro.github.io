Pathlib

Pathlib is a modern os.path, that aims to provide an intiative way to manipulate the path in operating system.

let introduce the basic syntax!

open a new folder say hello, then a file inside the file called world.py, with absoluate path /usr/yeah/hello/world.py, there is a file called sample.txt, that is

basic syntax

hello
-> world.py
-> sample.txt

in world.py

from pathlib import Path

path = Path('sample.txt')

print(path)
print(path.parts)       # '/', 'usr', 'yeah', 'hello', 'world.txt' 
print(path.name)        # sample
print(path.suffix)      # '.txt'
print(path.parent)      # 'usr', 'yeah', 'hello'
print(path.root)        # '/'

check the path

print(path.exists())    # True
print(path.is_dir())    # False
print(path.is_file())   # True

determine the user

print(path.home())      # /usr/yeah

pwd in shell => path.cwd(), cwd() => current working directory

absolute path

print(path.resolve())   # '/usr/yeah/hello/world.py'

note : if the path is not real, that means the file or folder is not exist, print the path and absolute path can be the same

path = Path('sample.txt')
print(path)             # sample.txt
print(path.resolve())   # /usr/yeah/hello/sample.txt

path = Path('notExist.txt')
print(path)             # notExists.txt
print(path.resolve())   # notExists.txt

create, remove

follow the shell command, mkdir

path = Path('yo.txt')
path.mkdir()

if we want to create a file that in an unexisted folder. Create the folder and file at the same time

path = Path('newFolder/lol.txt')
path.mkdir(parents=True, exist_ok=True)

parents=True means if the folder doesn’t exist, then create one exist_ok=True means if the file not exist, then create one. Otherwise, don’t throw exception

path.rmdir()

same as rm -r path

traverse directory

for file in path.iterdir():
    print(file, end=' ')

moreover, filter while traverse in two ways: (print all the .txt files)

for file in path.iterdir():
    if file.endswith('.txt'):
        print(file)

for file in path.glob('*.txt'):
    print(file)

glob() method except unix-style wildcard as parameter.

However, both method only traverse files in ‘./’ directory. We can traverse folder as well.

for file in path.rglob('*.txt'):
    print(file)

that means if any .txt file in directory, say ./lol/123.txt, it can find it out!

read, write

standard method

path = Path('sample.txt')

with open(path, 'r') as file:
    print(file.read())

pathlib methods

with path.open() as file:
    print(file.read())

or in one line

print(file.read_text())

write text

file.write_text('hello hell')

relative path

path = Path('sample.txt')
print(path.relative_to('/usr'))     # yeah/hello/sample.txt
